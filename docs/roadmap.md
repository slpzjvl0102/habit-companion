# Roadmap

## v1 — NOW: 8주 4-루틴 프로그램 (아주 작은 습관의 힘)
실험이 아니라 **내 아이 습관 만들기 도구.** 8주에 걸쳐 아주 작은 습관 1개부터 habit stacking으로 4개 루틴을 자리잡게 한다.

- **메커니즘(앱 내장):** 활성 습관이 5/7일 익으면 관리자가 다음 작은 습관을 *이어서* 추가 → 최대 **4개 체인.** = 『아주 작은 습관의 힘』 habit stacking.
- **동력:** 컴패니언(즉시 보상) + 관리자 승인. 잔소리 대신 환경.
- **cadence:** 습관 ~2주 → 8주 ≈ 4습관 (1–2주 ①5분 독서 / 3–4 +② / 5–6 +③ / 7–8 +④).
- **'성공':** 8주 후 4개 루틴이 스스로 돌고 잔소리가 줄어듦. (개인 도구 — 반증·대조군 같은 실험 장치 불필요.)
- **상태:** 빌드·테스트·배포 완료 (PWA 라이브: https://slpzjvl0102.github.io/habit-companion/). autoplan 리뷰의 버그 수정 반영(전날 승인·데이터 안전·컴패니언 진화·PIN 재잠금).

## v2 — 8주가 잘 굴러가면 / 더 키우고 싶으면
- **AI 말하는 컴패니언** [v1 빌드 세션에서 로그]: 펫이 성격 있는 격려 대사 생성·반응(예: Claude API). LLM 키는 클라이언트(PWA)에 못 넣음 → 작은 백엔드(서버리스) 필요.
- 원격/동기화 백엔드: 관리자가 *자기 폰* 에서 승인, 다기기 공유.
- 네이티브 APK(Android Studio) + 튼튼한 푸시 알림(저녁 리마인더).
- 컴패니언 깊이: 이름짓기·정체성, 테마 선택, 풍부한 진화; '고리' 시각 메타포.
- 습관 템플릿 라이브러리(나이·목표별); 실물 보상 자동화(용돈).
- 5개+ 루틴, 다자녀 등 확장.
